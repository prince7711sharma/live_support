🧪 Widget Demo Page

This page tests the Riddhi chat widget embed. Look for the chat bubble at the bottom right.

How to embed on YOUR website:

Add these 2 lines before </body>:

<script>
  window.RiddhiConfig = { apiUrl: 'http://localhost:8000' };
</script>
<script src="http://localhost:8000/widget/widget.js"></script>

Replace http://localhost:8000 with your deployed API URL when you go live.
